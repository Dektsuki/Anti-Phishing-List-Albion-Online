# English:

## Albion Online Anti-Phishing List

A community-driven project dedicated to identifying and blocking domains and URLs associated with phishing campaigns targeting Albion Online players.

## Purpose

Over the years, multiple fraudulent websites have appeared pretending to be legitimate Albion Online services, events, giveaways, guild recruitment pages, or support platforms in order to steal accounts, credentials, or sensitive information from players.

This repository maintains a public list of domains and URLs that have been identified as part of such phishing campaigns.

## Features

* Blocks known malicious domains.
* Automatically covers all subdomains of listed domains.
* Blocks specific URLs used in phishing campaigns.
* Public and transparent list.
* Continuously updated as new threats are identified.

## Repository Structure

* `List.json` → Main list used by scripts and security tools.

## Scope

The goal of this project is to help the Albion Online community identify and avoid known phishing infrastructure.

Domains and URLs included in this list are added only after verification and analysis.

## Reporting

If you believe a listed domain or URL is a false positive, or if you have information about a phishing campaign not yet included in this repository, please open an Issue.

## Disclaimer

This list is provided for informational and community protection purposes.

No list can guarantee complete protection against phishing attempts. Always verify website addresses carefully before entering credentials, downloading files, or providing sensitive information.

Use this information at your own discretion.

# Español:

# Albion Online Anti-Phishing List

Proyecto comunitario destinado a identificar y bloquear dominios y enlaces asociados a campañas de phishing relacionadas con Albion Online.

## Objetivo

Durante los últimos años han aparecido múltiples sitios fraudulentos que intentan hacerse pasar por servicios legítimos de Albion Online con el fin de robar cuentas, credenciales o información de los jugadores.

Este repositorio mantiene una lista pública de dominios y URLs detectados como parte de dichas campañas.

## Características

* Bloqueo de dominios maliciosos conocidos.
* Bloqueo automático de todos los subdominios de dichos dominios.
* Bloqueo de URLs específicas utilizadas en campañas de phishing.
* Lista pública y verificable.
* Actualización continua a medida que se identifican nuevas amenazas.

## Estructura

* `List.json` → Lista principal utilizada por herramientas y scripts.

## Aviso

La inclusión de un dominio o URL en esta lista implica que ha sido identificado como parte de una campaña de phishing o suplantación relacionada con Albion Online.

Si detectas un posible error o deseas aportar información adicional, puedes abrir un Issue en este repositorio.

## Descargo de responsabilidad

Esta lista se proporciona con fines informativos y de protección comunitaria. Siempre verifica manualmente cualquier sitio web antes de introducir credenciales o información sensible.
